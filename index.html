Autism Prediction LLM

This project leverages a large language model (LLM) to classify whether a patient is likely to be on the autism spectrum. Upon receiving the patient’s age as input, the LLM directs the case to a corresponding age-specific model—categorized into toddlers (1–3 years), children (4–11 years), teens (12–16 years), and adults (17–64 years). Each model performs classification based on a set of diagnostic inputs, which include A-score assessments gathered through targeted questions, as well as demographic and medical information such as age, gender, ethnicity, history of jaundice, and parental autism status.
